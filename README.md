## 启动
- 打开 .env.development 文件，修改后台接口：
  ```bash
  # 代理设置，可配置多个，不能换行，格式：[访问接口的根路径, 代理地址, 是否保持Host头]
   VITE_PROXY = [["/api","http://127.0.0.1:8080/api",false]]
  # 访问接口的根路径
   VITE_GLOB_API_URL =
- 运行、打包
   ```bash
   # 安装依赖
   yarn install
   # 开发环境运行
   yarn serve
   # 编译打包后运行访问
   yarn preview
   # 打包
   yarn build
